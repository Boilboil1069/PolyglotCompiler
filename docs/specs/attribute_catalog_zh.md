# 属性目录（v1.16.0 起）

本目录列出 Ploy 前端内建注册表识别的全部注解。注册表外的注解以 sema
警告形式接受，便于第三方工具在不改动编译器的情况下扩展。

## 调用形式

```
@<name>                       # 无参数
@<name>(<arg>, <arg>, ...)    # 逗号分隔，按字串原样捕获
```

多个注解可叠加并按从左到右的顺序生效；与可见性关键字
（`PUB` / `PRIVATE`）的相对顺序不影响语义。

## 目录

### 内联

| 名称              | 参数 | 含义                                    |
|-------------------|------|-----------------------------------------|
| `@inline`         | —    | 提示：每个调用点优先内联。              |
| `@noinline`       | —    | 提示：禁止内联。                        |
| `@always_inline`  | —    | 强制内联；sema 接受，优化器待接入。     |

### 代码布局 / Profiling

| 名称              | 参数 | 含义                                    |
|-------------------|------|-----------------------------------------|
| `@hot`            | —    | 热路径优化（偏向速度）。                |
| `@cold`           | —    | 冷路径优化（偏向体积）。                |
| `@profile`        | —    | 始终生成 profile 采样桥。               |
| `@no_profile`     | —    | 抑制 profile 采集。                     |

### 文档 / 链接

| 名称              | 参数                              | 含义                                          |
|-------------------|-----------------------------------|-----------------------------------------------|
| `@deprecated`     | `("msg")`                         | 标记符号已弃用；sema 在使用点警告。           |
| `@link_name`      | `("symbol")`                      | 覆盖 mangle 后的导出名。                      |
| `@target`         | `("x86_64,arm64,...")`            | 限定符号可用的架构。                          |

## 注解可出现的位置

v1.16.0 MVP 仅在 `FUNC`、`ASYNC FUNC`、`STRUCT` 前接受该前缀；其他
顶层形式（`CLASS`、`CONST`、`TYPE`、`MAP_FUNC`、`EXTEND` 等）由解析器
报告诊断。把前缀扩展至这些形式记入
`docs/realization/visibility_attrs_zh.md` 的后续工作。

## 跨语言映射（计划）

MVP 暂不把注解传播到宿主语言绑定。接入后预期的映射为：

| Ploy 属性          | C++                          | Rust                       | Java / .NET / Python   |
|--------------------|------------------------------|----------------------------|------------------------|
| `@inline`          | `inline`                     | `#[inline]`                | （宿主默认提示）       |
| `@always_inline`   | `[[gnu::always_inline]]`     | `#[inline(always)]`        | （宿主默认提示）       |
| `@noinline`        | `[[gnu::noinline]]`          | `#[inline(never)]`         | （宿主默认提示）       |
| `@deprecated("m")` | `[[deprecated("m")]]`        | `#[deprecated = "m"]`      | 宿主原生等价物         |
| `@link_name("s")`  | 链接器 `--defsym` / `extern` | `#[export_name = "s"]`     | 宿主原生等价物         |
